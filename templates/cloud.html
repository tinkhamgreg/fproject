The Use of The Cloud In Continuous Development & Deployment

Utilizing the cloud for continuous development and deployment has a lot of benefits over just local work. The first and foremost is the offshoring of resources that are needed for things such as docker builds or worrying about the upkeep of a system instead of utilizing Amazon Web Services. Secondly, the online repository on Github in this instance can give the feel of a central repository for multiple members of a project to from and look at.
